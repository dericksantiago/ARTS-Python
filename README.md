# ARTS - Accounts Receivable Tracking System
### Python Edition v1.0

A full migration of a legacy **Clipper Summer '87** DOS-based 
Accounts Receivable and Container Tracking System to modern Python.

---

## 🚀 Project Overview

| Item | Detail |
|------|--------|
| **Original Platform** | Clipper Summer '87 / DOS / DOSBox |
| **New Platform** | Python 3.13 / SQLite |
| **Industry** | Freight & Container Logistics |
| **Database** | 26 DBF files → Single SQLite database |
| **Records** | 115,000+ real business records migrated |
| **Status** | Active development |

## 🚦 How to Run

```bash
# Clone the repository
git clone https://github.com/dericksantiago/ARTS-Python.git

# Navigate to project folder
cd ARTS-Python

# Install dependencies
pip install colorama

# Run the application
python run_arts.py
```
